Propuesta de Proyecto TecWeb

Tengo como propuesta crear una página web para un GYM llamado Space Sport GYM, este gimnacio requiere un pagina web donde los clientes puedan observar los aparatos de trabajo asi como el uso que se les puede dar, tambíen algunas rutinas dependiendo de lo que desean realizar si es día de espalda, pecho, ó piernas, lo primero que se tendra que hacer para acceder a todo esto es crear un usuario y su contrasenia así tendra información a todo lo mencionado, asi como tambien el usuario podra dejar su comentario hacerca del gym, el usuario podra inscribirse desde aquí y hacer el pago desde la comodidad de su casa, tambien aparecera una apartado donde se le indica a los usuarios la fecha de la mensualidad etc, tendremos otro apartado donde los usuarios podran hacer sus pedidos de sus suplementos y comprarlos o solo apartarlos.
