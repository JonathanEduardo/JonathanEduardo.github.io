Selectores

Tipos de selectores

Selector Universal
Aplica un estilo a todos los elementos del documento ejemplo: *{font-family:Arial;}



Selector de Tipo ETIQUETA
Aplica un estilo a todos los elementos de la etiqueta ejemplo: h1 {font-size : 30px;}



Selector De Clase
Aplica un estilo a todos los elementos que tengan el atributo class especificado por un valor ejemplo: .nota {color: green;} -> (p class = "nota")



Selector de ID
Aplica el estilo a los elementos que tengan el atributo ID con un valor dado ejemplo: #Introduccion{background-color: cyan;} div id="Introduccion"

Selector Hijo Aplica el estilo a los elementos que son hijos inmediatos de otro ejemplo: li > a{font-weigth: bold;}

Selector Descendiente Aplica el estilo a los elementos que son hijo pero no necesariamente directos ejemplo: p a{font-weigth: bold;}
